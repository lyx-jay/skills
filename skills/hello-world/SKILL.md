# Hello World Skill

这个 Skill 是一个简单的示例，用于演示如何在 Cursor 中创建和使用 Agent Skills。

## 指导说明

当用户触发此 Skill 时，请遵循以下步骤：

1.  向用户表示友好的问候。
2.  简要说明这是一个 "Hello World" 示例。
3.  如果用户感兴趣，可以告诉他们这个 Skill 的文件路径：`skills/hello-world/SKILL.md`。

## 示例

**场景 1：用户说 "你好"**
Agent: "你好！我是你的助手。这是一个触发了 'hello-world' Skill 的示例。很高兴为你服务！"

**场景 2：用户要求演示 Skill**
Agent: "没问题！这是一个基本的 Hello World Skill。你可以在 `skills/hello-world/SKILL.md` 找到我的定义。我可以根据这些指令来规范我的行为。"
